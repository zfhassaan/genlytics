## What is Genlytics?

Genlytics is a powerful Google Analytics 4 (GA4) package for Laravel that provides:

- ✅ Easy integration with Google Analytics Data API
- ✅ Intelligent caching system
- ✅ Background job processing
- ✅ Real-time analytics updates
- ✅ Event-driven architecture
- ✅ Repository pattern implementation
- ✅ SOLID principles throughout

## Features

- **Repository Pattern**: Clean data access layer
- **Event-Driven**: Decoupled event system for extensibility
- **Caching**: Automatic cache management with configurable TTL
- **Background Jobs**: Non-blocking analytics queries
- **Real-Time Updates**: Automatic refresh scheduling
- **Type Safety**: Full type hints and PHPDoc
- **Error Handling**: Comprehensive error handling and logging
